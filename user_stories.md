# User Stories

## Admin User Stories

**US-01: Admin Login**
As an admin, I want to log in to a secure dashboard so that I can manage the system and its users.

**US-02: Manage Doctors**
As an admin, I want to add, update, and remove doctor accounts so that the system always reflects the current medical staff.

**US-03: Manage Patients**
As an admin, I want to view and manage patient records so that I can ensure data accuracy and handle account issues.

**US-04: View All Appointments**
As an admin, I want to view all appointments across the system so that I can monitor scheduling activity and resolve conflicts.

**US-05: Manage System Access**
As an admin, I want to assign roles and permissions to users so that only authorized personnel can access sensitive features.

**US-06: View Dashboard Analytics**
As an admin, I want to see summary statistics (total patients, doctors, appointments) on my dashboard so that I can quickly assess system activity.

## Patient User Stories

**US-07: Patient Registration**
As a patient, I want to register an account so that I can access the appointment booking system.

**US-08: Patient Login**
As a patient, I want to log in to my account so that I can manage my appointments and view my medical information.

**US-09: Book an Appointment**
As a patient, I want to book an appointment with an available doctor so that I can receive medical care at a convenient time.

**US-10: View My Appointments**
As a patient, I want to view my upcoming and past appointments so that I can keep track of my medical schedule.

**US-11: Cancel an Appointment**
As a patient, I want to cancel an appointment I no longer need so that the time slot becomes available for other patients.

**US-12: View Prescriptions**
As a patient, I want to view prescriptions issued to me so that I can follow my treatment plan correctly.

**US-13: Update My Profile**
As a patient, I want to update my personal information so that my records remain current and accurate.

## Doctor User Stories

**US-14: Doctor Login**
As a doctor, I want to log in to my dashboard so that I can manage my schedule and patient appointments.

**US-15: View My Appointments**
As a doctor, I want to view my scheduled appointments so that I can prepare for upcoming patient visits.

**US-16: Manage Availability**
As a doctor, I want to set and update my available time slots so that patients can only book when I am free.

**US-17: Issue a Prescription**
As a doctor, I want to create a prescription for a patient so that they receive the correct medication and dosage instructions.

**US-18: View Patient History**
As a doctor, I want to view a patient's appointment and prescription history so that I can make informed medical decisions.

**US-19: Update Appointment Status**
As a doctor, I want to mark appointments as completed or cancelled so that my schedule accurately reflects my activity.

**US-20: Update My Profile**
As a doctor, I want to update my professional details and contact information so that patients and admins have accurate records.
